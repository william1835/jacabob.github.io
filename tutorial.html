Basics - Input, Process, Output

All pieces of code for any language take an Input, processes it then outputs it. In python it is very easy.

First lets look at outputs. In python to output somethiong that the user can see on the screen we use print(). Remember that print must all be in lower case letters and the brackets must be there. You may notice that this isn't going to do anything as there isn't any text or data in the brackets.

Here are some examples:

we can print out data that the user has entered into the script. First the user must create a variable and set it as an input like this. In the example below a is stored as an input. This a is a variable and it can be called anything you like.

print(a) is calling the function a and outputing what a is equal to. So if a=2 then, 2 would be shown to the user.

To get an input from the user we must first create a variable. This is a simple as typing in a word such as 'b' and set it equal to something. This is what a variable is.
